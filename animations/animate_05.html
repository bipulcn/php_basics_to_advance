CSS Animations

CSS3 animations allows animation of most HTML elements without using JavaScript or Flash!

Note: This example does not work in Internet Explorer 9 and earlier versions.

Hello world

When you specify CSS styles inside the @keyframes rule, the animation will gradually change from the current style to the new style at certain times. To get an animation to work, you must bind the animation to an element. The following example binds the "example" animation to the <h3> element. The animation will last for 4 seconds, and it will gradually change the background-color of the <h3> element from "red" to "yellow":
	h3 {
	    width: 100px;
	    height: 100px;
	    padding: 10px;
	    background-color: red;
	    animation-name: example;
	    animation-duration: 4s;
	}

	/* Standard syntax */
	@keyframes example {
	    from {background-color: red;}
	    to {background-color: yellow;}
	}
