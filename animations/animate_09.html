CSS Animations

CSS3 animations allows animation of most HTML elements without using JavaScript or Flash!

Note: This example does not work in Internet Explorer 9 and earlier versions.

linear

ease

ease-in

ease-out

ease-in-out

It is also possible to use percent. By using percent, you can add as many style changes as you like.The following example will change the background-color of the <h3> element when the animation is 25% complete, 50% complete, and again when the animation is 100% complete:
	h3 {
	    width: 100px;
	    height: 50px;
	    padding: 10px;
	    background-color: orange;
	    font-weight: bold;
	    position: relative;
	    animation: mymove 5s infinite;
	    animation-direction: alternate;
	}

	/* Standard syntax */
	#div1 {animation-timing-function: linear;}
	#div2 {animation-timing-function: ease;}
	#div3 {animation-timing-function: ease-in;}
	#div4 {animation-timing-function: ease-out;}
	#div5 {animation-timing-function: ease-in-out;}

	/* Standard syntax */
	@keyframes mymove {
	    from {left: 0px;}
	    to {left: 80%;}
	}
