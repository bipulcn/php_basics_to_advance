CSS Animations

CSS3 animations allows animation of most HTML elements without using JavaScript or Flash!

Note: This example does not work in Internet Explorer 9 and earlier versions.

It is also possible to use percent. By using percent, you can add as many style changes as you like.The following example will change the background-color of the <h3> element when the animation is 25% complete, 50% complete, and again when the animation is 100% complete:
	
	h3 {
		padding: 20px;
		height: 60px;
		width: 100px;
	    background-color: red;
	    position: relative;
	}
	.reverse {
	    animation-name: example;
	    animation-duration: 4s;
	    animation-iteration-count: infinite;  
	    animation-direction: reverse;
	}
	.regular {		
	    animation-name: example;
	    animation-duration: 4s;
	    animation-iteration-count: infinite;  
	}
	.alternat {		
	    animation-name: example;
	    animation-duration: 4s;
	    animation-iteration-count: infinite;  
	    animation-direction: alternate;
	}

	/* Safari 4.0 - 8.0 */
	@-webkit-keyframes example {
	    0%   {background-color:red; left:0px; top:0px;}
	    25%  {background-color:yellow; left:200px; top:0px;}
	    50%  {background-color:blue; left:200px; top:200px;}
	    75%  {background-color:green; left:0px; top:200px;}
	    100% {background-color:red; left:0px; top:0px;}
	}
